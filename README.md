# Algoritmos e estruturas de dados I

## Sobre

Este repositório será utilizado para armazenar os arquivos de trabalhos da
disciplina de Algoritmos e estruturas de dados I do terceiro semestre de Ciência
da Computação da Universidade Federal de Pelotas.

## Misc

Para testar um algoritmo, é necessário entrar na pasta ```tests```.

```
cd tests
```

O arquivo ```main.c``` estará na pasta ```src```.

Para compilar e executar o arquivo:

```
make run
```

Para compilar e executar o arquivo com o Valgrind:

```
make valgrind
```
